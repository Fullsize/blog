**计算3-3**

先转为二进制

```c
// 3
0000 0011

// -3
1000 0011
```

反码

```c
// -3
1000 0011
// 反码后
1111 1100

```

补码(加1)

```c
1111 1100
+
0000 0001
=
1111 1101 
```

加上补码

```c
0000 0011
+
1111 1101
=
0000 0000
```
