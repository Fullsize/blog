### 基本模版

```javascript
function backtrack(路径, 选择列表) {
    if (满足结束条件) {
        记录结果
        return;
    }
    
    for (选择 in 选择列表) {
        做出选择
        backtrack(更新路径, 更新选择列表)
        撤销选择
    }
}

```
