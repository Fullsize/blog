## 同一浏览器，不同tab清除

**利用localStorage的storage的通知，使其他tab强制刷新**

```javascript
<!--
 * @Author: Fullsize
 * @Date: 2021-10-21 16:11:48
 * @LastEditors: Fullsize
 * @LastEditTime: 2021-11-05 14:30:22
 * @FilePath: /test/index.html
-->
<!DOCTYPE html>
<html lang="en">

<head>
	<meta charset="UTF-8">
	<meta http-equiv="X-UA-Compatible" content="IE=edge">
	<meta name="viewport" content="width=device-width, initial-scale=1.0">
	<link rel="stylesheet" href="./index.css">
	<title>Document</title>
</head>

<body>
	<button onclick="changeVersion()">修改版本</button>
	<h2 id="text"></h2>
</body>
<script>
	document.getElementById('text').innerText = localStorage.getItem('version')
	function changeVersion() {
		localStorage.setItem('version', new Date().getTime())
		document.getElementById('text').innerText = localStorage.getItem('version')
	}
	window.addEventListener("storage", function (e) {
		console.log(e)
	});
</script>

</html>
```
